# m_amazon_dash
Title
Amazon Sales Analysis Dashboard | Power BI
Description

This project presents an interactive Amazon Sales Analysis Dashboard developed in Power BI to analyze sales performance, product trends, and customer engagement metrics. The dashboard provides a comprehensive view of sales data through dynamic visualizations, KPI cards, and interactive filters, helping users gain valuable business insights.
Key Features

📊 Sales Analysis by Month

Implemented an Area Chart to visualize month-wise sales trends.
Helps identify seasonal patterns, sales growth, and peak-performing months.

📈 Sales Analysis by Week

Created a Column Chart displaying week-number-wise sales performance.
Enables detailed tracking of weekly sales fluctuations and trends.

📌 KPI Cards

YTD (Year-to-Date) Sales
QTD (Quarter-to-Date) Sales
Total Reviews
YTD Product Sales
Provides a quick overview of business performance metrics.

🎯 Interactive Slicers

Product Category Slicer
Quarter (QTR) Slicer
Allows users to filter data dynamically and analyze sales across different categories and time periods.

📦 Product Performance Analysis

Top 5 Products by YTD Sales
Top 5 Products by YTD Reviews
Product Category-wise Sales Breakdown
Technical Implementation

✔ Data Cleaning & Transformation

Processed and cleaned raw sales data using Power Query.
Handled missing values, data formatting, and data consistency checks.

✔ Data Modeling

Designed relationships between tables to build an optimized data model.
Implemented a star-schema approach for efficient reporting.

✔ DAX Measures

Created custom DAX measures for:
YTD Sales
QTD Sales
Total Reviews
Sales Growth Analysis
Dynamic KPI Calculations

✔ Data Visualization

Developed interactive and user-friendly dashboards using Power BI visualizations.
Applied formatting, themes, and responsive layouts for enhanced user experience.
Tools & Technologies
Power BI
Power Query
DAX (Data Analysis Expressions)
Data Modeling
Data Visualization
Excel/CSV Dataset
Project Outcome

This dashboard helps stakeholders monitor Amazon product sales performance, identify top-selling categories and products, track customer reviews, and make data-driven business decisions through interactive and insightful visual analytics.
